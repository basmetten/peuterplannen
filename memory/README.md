# Memory Index

This directory contains structured long-term memory for Bassistent.

## Structure
- `preferences.md`: User preferences, habits, and communication style.
- `contacts.md`: Important people and relationship contexts.
- `projects.md`: Ongoing work, goals, and status updates.
- `facts.md`: General facts about Bas's life (home, office, tech setup).
- `archive/`: Compressed/summarized older memories.

## Usage
- Use `memory_search` to find relevant snippets.
- Use `memory_get` to read specific files.
- Update files when new persistent info is learned.
